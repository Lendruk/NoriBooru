# NoriBooru
WIP self-hosted media database and management system.
**NOT PRODUCTION READY**

## Why?
I want to learn sveltekit and this seems like an interesting idea to execute.

## How to run

### Locally
In the project root directory run:
```sh
npm run prepare:dev
```
And
```sh
npm run dev
```

### Docker
In the project root directory run:
```sh
docker-compose up
```

## Requirements
- ffmpeg
